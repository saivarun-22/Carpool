#  Carpool 2026.

This Project is inspired from the Existing Carpool idea with moderate enhancements in the design and personalized fare calculation algorithms and functions compared
with the other carpool websites, and part of this project was developed with assistance of AI tools. and this project is developed using the current technologies we 
are familiar with, such as flask,sql,googlecloudapi(Openauth),etc.



### Frontend
- HTML5
- CSS3
- JavaScript
- Jinja2 (for templating)
- Bootstrap

### Backend
- Python (Flask Framework)
- Openauth googlecloudapi (for google authentication in website)

### Database
- MySQL

## Installation Steps ##

1. git clone https://github.com/saivarun-22/carpool.git
2. cd carpool
3. python -m venv venv
4. venv\Scripts\activate      
5. pip install -r requirements.txt   # consists of libraries
6. python3 app.py # to run the Main file
